# Language Policy: English Core, Thai Soul

This repository uses English as the primary language for:

- Source code comments and docstrings
- Documentation and reports
- README and release artifacts

## Preserved Brand Terms

The following terms are intentionally preserved in English form:

- `NaMo`
- `Dhammic Moat`
- `Karuna Protocol`

## Editorial Rules

- Prefer clear engineering English over mixed-language sentences.
- Keep legal and compliance wording explicit and auditable.
- Avoid encoding artifacts and mojibake in committed files.
- Use ASCII where possible unless technical notation requires otherwise.
